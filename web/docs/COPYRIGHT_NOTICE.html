Foundational Concepts

  • Storage-less Computing: No persisted bits/bytes. Observables exist only when extracted from geometry via deterministic rules.
  • Geometric Substrates: Manifolds (e.g., Swartz Diamond Lattice) provide topology and invariants that embody information intrinsically.
  • Extraction Rules: Morphisms mapping geometric properties (local frames, curvature, adjacency, winding, symmetries) to computable data at runtime.
  • Determinism and Reproducibility: Given a manifold identity and a rule identity, extraction yields the same result; no hidden mutable state.
