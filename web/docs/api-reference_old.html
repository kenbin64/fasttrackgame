Substrate class

Base class for all data containers. Substrates hold primitives (data) and operations (actions).

from helix import Substrate

class MySubstrate(Substrate):
    def __init__(self, name: str):
        super().__init__(name)
        self.register_primitive("color", RGB(255, 100, 50))
        self.register_operation("reset", self._reset)
    
    @property
    def domain(self) -> str:
        return "myapp.data"
    
    def _reset(self):
        self.set_primitive("color", RGB(0, 0, 0))

SecureResourceLocator class

Singleton registry for all SRL mappings. Provides O(1) lookup for any registered resource.

from helix import SecureResourceLocator

SRL = SecureResourceLocator()

# Register
address = SRL.register(my_substrate)

# Lookup
value = SRL.resolve("srl://domain/substrate/primitive")

# Search
matches = SRL.find("user.*profile")

# List all
SRL.list_substrates()
MethodParametersReturnsDescription
register(substrate) substrate: Substrate str Register substrate, return SRL address
unregister(address) address: str bool Remove substrate from registry
resolve(address) address: str Any Get primitive value at address
invoke(address, *args) address: str, args Any Execute operation at address
find(pattern) pattern: str (regex) list[str] Find matching addresses
list_substrates() None list[str] List all registered substrates
exists(address) address: str bool Check if address is registered
get_substrate(address) address: str Substrate Get substrate object

Top-Level Functions functions

Convenience functions for working with SRLs.

srl(address) â†’ Any

Resolve an SRL address to its value. O(1) lookup.

from helix import srl

color = srl("srl://graphics.color/palette/primary")
speed = srl("srl://vehicle.car/tesla/speed")

Returns: The primitive value at the address

invoke(address, *args, **kwargs) â†’ Any

Execute an operation at an SRL address.

from helix import invoke

result = invoke("srl://vehicle.car/tesla/accelerate", 10)
user = invoke("srl://database.users/profiles/find", id=42)

Returns: The result of the operation

register_substrate(substrate) â†’ str

Register a substrate with the global SRL registry.

from helix import register_substrate

address = register_substrate(my_substrate)
print(address)  # â†’ "srl://domain/name"

Returns: The full SRL address of the registered substrate
