Overview

The ButterflyFX Helix SDK provides a dimensional computing kernel for building applications with geometric data structures. The core concepts are:

  • HelixKernel ‚Äî State machine for dimensional navigation
  • Substrate ‚Äî Manifold where tokens (entities) exist
  • Tokens ‚Äî Entities with location, signature, and payload
  • Levels 0-6 ‚Äî Dimensional layers from potential to whole

Authentication

API requests require authentication via API key in the header:

# HTTP Header
Authorization: Bearer bfx_live_your_api_key_here

# Python
from butterflyfx import HelixClient

client = HelixClient(api_key="bfx_live_your_api_key")

// JavaScript
import { HelixClient } from '@butterflyfx/helix';

const client = new HelixClient({ apiKey: 'bfx_live_your_api_key' });

Get your API key from the Developer Portal.

HelixKernel

The core state machine for dimensional navigation. Maintains state as (spiral, level) pair.

Constructor

HelixKernel(substrate: Substrate | None = None)
ParameterTypeDefaultDescription
substrate Substrate | None None Optional substrate to attach for materialization

Example

from helix import HelixKernel, Substrate

# Without substrate
kernel = HelixKernel()

# With substrate
substrate = Substrate()
kernel = HelixKernel(substrate=substrate)

invoke(level)

Jump to a specific dimensional level within the current spiral.

invoke(level: int) ‚Üí HelixState
ParameterTypeDescription
level required int (0-6) Target dimensional level
ReturnsDescription
HelixState New state after invocation
RaisesWhen
ValueError Level not in range 0-6

Example

kernel = HelixKernel()
print(kernel.state)  # (0, 0:Potential)

kernel.invoke(3)
print(kernel.state)  # (0, 3:Width)

kernel.invoke(6)
print(kernel.state)  # (0, 6:Whole)

spiral_up() / spiral_down()

Move to adjacent spirals in the helix.

spiral_up() ‚Üí HelixState # Requires level=6
spiral_down() ‚Üí HelixState # Requires level=0
MethodPreconditionTransition
spiral_up() level == 6 (s, 6) ‚Üí (s+1, 0)
spiral_down() level == 0 (s, 0) ‚Üí (s-1, 6)

Example

kernel = HelixKernel()
kernel.invoke(6)      # Must be at level 6
kernel.spiral_up()
print(kernel.state)   # (1, 0:Potential)

kernel.spiral_down()
print(kernel.state)   # (0, 6:Whole)

collapse()

Return to potential (level 0) within current spiral. Idempotent.

collapse() ‚Üí HelixState

Example

kernel.invoke(5)
kernel.collapse()
print(kernel.state)   # (0, 0:Potential)

# Idempotent - calling again has no effect
kernel.collapse()
print(kernel.state)   # (0, 0:Potential)

materialize()

Get tokens at current state from attached substrate.

materialize() ‚Üí Set[Token]
ReturnsDescription
Set[Token] Tokens materialized at current (spiral, level)

Example

substrate = Substrate()
substrate.add_token("entity1", spiral=0)

kernel = HelixKernel(substrate)
kernel.invoke(6)

tokens = kernel.materialize()
print(tokens)  # {Token(entity1, level=6)}
